# Azkar Website

A simple and interactive web application that displays daily **Azkar** (Islamic supplications) for morning and evening, includes **Surah Al-Kahf**, and a **Dhikr counter**.

---

## Features

- **Morning & Evening Azkar**: View daily supplications for morning and evening.
- **Surah Al-Kahf**: Read Surah Al-Kahf directly on the website.
- **Dhikr Counter**: Keep track of your recitations with an interactive counter.
- **Responsive Design**: Works on desktop and mobile devices.
- **Clean UI**: Simple and user-friendly interface with easy navigation.
